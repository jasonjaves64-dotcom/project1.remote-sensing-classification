## 二、模块功能详解

| 阶段 | 模块名称 | 核心功能 | 关键技术 | 输出 |
|------|----------|----------|----------|------|
| 1 | SpatialAligner | 空间配准 | 双线性插值 | 统一分辨率数据 |
| 2 | CloudDetector | 云检测 | NDVI + SAR联合检测 | 云掩膜 |
| 3 | TemporalInterpolator | 时序插值 | 线性插值 | 填充后时序数据 |
| 4 | SpectralNormalizer | 光谱归一化 | Z-score | 标准化数据 |
| 5 | DataAugmenter | 数据增强 | 空间/时序变换 | 增强样本 |
| 6 | DataQualityChecker | 质量检查 | 多维度验证 | 过滤后样本 |

## 三、关键设计特点

### 1. 多模态协同处理
- 光学、SAR、DEM三模态同步处理
- 空间变换同步应用于所有模态和标签
- 确保数据一致性

### 2. 可配置性
```yaml
preprocess:
  target_resolution: 10.0    # 目标分辨率(米)
  cloud_threshold: 0.3       # 云检测阈值
  max_gap: 30                # 最大插值间隔(天)
  normalize: true            # 是否归一化
  augment: true              # 是否增强
  augment_prob: 0.5          # 增强概率
```

### 3. 鲁棒性设计
- 质量检查机制过滤低质量样本
- 边界条件处理（首末帧保持）
- 统计量持久化避免重新计算

### 4. 可扩展性
- 模块化设计，易于添加新处理步骤
- 支持自定义数据格式
- 灵活的配置接口

## 四、技术优势

| 方面 | 传统方法 | 本框架 |
|------|----------|--------|
| 多源数据对齐 | 手动配准 | 自动化空间配准 |
| 云遮挡处理 | 简单掩码 | 智能插值填充 |
| 数据增强 | 单一变换 | 多模态同步增强 |
| 质量控制 | 人工检查 | 自动化质量过滤 |
| 可重复性 | 低 | 高（统计量持久化） |

## 六、使用示例

# 创建配置
config = PreprocessConfig(
    target_resolution=10.0,
    cloud_threshold=0.3,
    augment=True
)

# 创建管道
pipeline = PreprocessPipeline(config)

# 处理数据
sample = pipeline.process(
    raw_data={'opt': opt_data, 'sar': sar_data, 'dem': dem_data},
    transforms=transform_params,
