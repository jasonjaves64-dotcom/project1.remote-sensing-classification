02FOUNDATIONS

Mathematical Infrastructure

Evidential Deep Learning — Dirichlet Prior

V6 inherits V5EDL's EDL framework. The network outputs concentration parameters α = [α₁, ..., αK] for each pixel, parameterizing a Dirichlet distribution over K=7 crop classes:

Dir(p | α) = Γ(α₀) / Πₖ Γ(αₖ) · Πₖ pₖαₖ⁻¹
where α₀ = Σₖ αₖ is the precision (total evidence),
    pₖ = αₖ / α₀ is the expected probability for class k

EDL Loss Function

L(α, y) = LCE + λ(t) · KL[ Dir(p|𝛼) || Dir(p|1) ]
Cross-entropy on Dirichlet expectation + KL divergence to uniform prior
λ(t) = λmax · min(1, t / Tanneal) — linear annealing from 0 to 0.5 over 50 epochs

TemporalLite — Efficient Temporal Aggregation

Replaces the per-pixel TransformerEncoder (O(n²) self-attention across T timesteps) with lightweight 1D depthwise convolutions:

ht = DWConv1D(xt-k:t+k, kernel=k=3) · σ(Gate(x))
Complexity: O(T) vs O(T²) for self-attention
3 instances: s1 (H scale, 64ch), s2 (H/2, 128ch), opt_p2 (H/2, 256ch)
Total params: 2,243 — negligible overhead for ~48× speedup at T=24

Multi-Scale Cross-Modal Attention

Optical and SAR features interact at three spatial resolutions via CrossModalLite:

CrossModalLite(Qopt, Ksar) = Qopt + MLP(Attn(Qopt, Ksar, Ksar))
 where Attn(Q,K,V) = softmax(QKT / √d) · V
H scale: opt_p2(256ch) → 64ch projection ↔ sar_s1(64ch) — CrossModalLite(64, n_heads=1)
H/2 scale: opt_p2(256ch) → 128ch projection ↔ sar_s2(128ch) — CrossModalLite(128, n_heads=4)
H/4 scale: original CrossModalAttention(feat_dim=512, n_heads=16) — inherited from V5EDL
03BLOCKS

Six Enhancement Blocks

All blocks guarded behind use_v6_enhancements flag. V5EDL stays completely clean.

TemporalLite

2,243 PARAMS · 48× SPEEDUP

DepthwiseConv1D (k=3) replaces TransformerEncoder self-attention. Preserves temporal dynamics while eliminating O(n²) complexity. Applied to s1, s2, and opt_p2 streams.

Early Fusion

2,560 PARAMS

ModalNormalize (instance-level standardization) + 1×1 Conv2d: 20 channels → 128 channels. Creates unified representation before backbone processing. Future work: wire as cross-attention Q-source.

DEM 5-Path Injection

568,128 PARAMS

Five interaction paths: (1) DEMEncoder, (2) DEMOpticalConditioner (FiLM), (3) DEMTemporalProj (elevation-phenology bias), (4) DEMSpatialConditioner, (5) Decoder skip connection.

Multi-Scale Cross-Attn

525,760 PARAMS

CrossModalLite at H (64ch), H/2 (128ch), and original H/4 (512ch) CrossModalAttention. Three fusion resolutions capture fine, medium, and coarse feature interactions.

Multi-Task Heads

57,831 PARAMS

LAIRegressionHead (1 output), GrowthStageHead (5 classes), BoundaryHead (1 binary). Pseudo-label generators for self-training. Combined via MultiTaskLoss (learnable task weights, 5 tasks).

LightSceneHead

19,467 PARAMS

Scene classification (4 types: annual crop, permanent, grassland, other) + crop distribution (7-class mixture). Lightweight global pooling + FC. Enables parcel-level scene understanding.

04ISOLATION
