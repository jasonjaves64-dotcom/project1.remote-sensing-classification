一、项目概述
-----------------------------------------------------------
本项目旨在开发一套基于深度学习的遥感影像作物分类系统，
融合光学、SAR和DEM多源数据，实现高精度的作物类型识别，
并集成不确定性估计能力，提升模型决策可靠性。

四、技术亮点
-----------------------------------------------------------

1. **时序动态特征建模**
   - **时间编码器改进**：采用时间维度加权平均策略，正确聚合所有时间步特征，避免原始版本只取第一个时间步导致的动态信息丢失问题
   - **云掩膜加权机制**：支持根据云掩膜权重自动跳过被遮挡的时间步，只利用高质量观测数据进行特征提取
   - **物候特征建模**：通过DOY（日序）编码将时间信息注入模型，增强对作物生长周期的建模能力

2. **DEM几何约束注入**
   - **FiLM调制技术**：采用Feature-wise Linear Modulation将DEM几何特征作为条件信息注入Transformer编码器的每一层
   - **分层注入策略**：DEM特征在时间编码器的每个Transformer层都进行条件调制，确保几何约束贯穿整个特征提取过程
   - **空间条件感知**：DEM高度信息通过FiLM参数动态调整特征的仿射变换，使模型能够感知地形变化对作物分布的影响

3. **内存优化与计算效率**
   - **梯度检查点技术**：在Transformer编码器和解码器中设置44个梯度检查点，通过牺牲少量计算时间换取约50%的显存占用减少
   - **混合精度训练**：支持FP16混合精度训练，进一步降低显存消耗并提升训练速度
   - **窗口注意力机制**：将全局自注意力替换为窗口注意力（window attention），避免高分辨率输入时的OOM问题，同时保持局部特征建模能力

4. **多模态融合机制**
   - **三模态协同架构**：光学（10通道，含6个原始波段+4个植被指数）、SAR（5通道）、DEM（5通道）三模态深度融合
   - **跨模态注意力**：设计专门的跨模态注意力模块，实现光学与SAR特征的深度交互
   - **自适应门控融合**：采用自适应权重机制动态调整各模态特征的贡献，根据输入数据质量自动分配权重

5. **EDL-Ensemble不确定性估计**
   - **证据深度学习（EDL）**：使用Dirichlet分布建模类别证据，输出概率分布参数α而非单一预测
   - **KL正则化损失**：引入KL散度正则化项约束证据分布，防止模型过度自信
   - **三种不确定性度量**：
     - **Vacuity（数据不确定性）**：K/S，衡量证据不足程度，反映数据质量
     - **Dissonance（认知不确定性）**：1-Σp²，衡量类间证据冲突，反映模型知识边界
     - **Class Variance（类别方差）**：p(1-p)/(S+1)，衡量每类预测的不确定性
   - **Ensemble策略**：支持多轮推理和测试时增强（TTA），通过证据级融合提升不确定性估计的可靠性
