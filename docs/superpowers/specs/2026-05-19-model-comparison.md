## 三、性能指标对比（文献报告值）

| 模型 | 数据集 | mIoU / OA | 备注 |

## 四、FusionCropNet 独有功能（竞品缺失）

| 功能 | 独有性 | 价值 |
|------|--------|------|
| **EDL 不确定性估计** | 🔴 独有 | 量化预测可信度，分离偶然/认知不确定性 |
| **三模态融合 (Opt+SAR+DEM)** | 🔴 独有 | SAR 全天候 + DEM 地形先验，其他模型最多双模态 |
| **跨模态交叉注意** | 🔴 独有 | 光学↔SAR 双向注意，而非简单拼接 |
| **缺失模态鲁棒性** | 🔴 独有 | 模态 Dropout 训练，单一模态降级可用 |
| **MIL 弱监督** | 🔴 独有 | Bag 级标注即可训练，降低标注成本 |
| **HPO 超参搜索** | 🔴 独有 | Ray Tune 集成，自动优化超参 |
| **NDVI 物候辅助** | 🔴 独有 | 物理先验增强时序理解 |

## 五、FusionCropNet 缺失功能（竞品已有）

| 功能 | 差距 | 引入价值 | V6 计划 |
|------|------|---------|---------|
| **SSL 预训练** | 高 | 标签效率 2-3× | ✅ Phase 3 |
| **基础模型骨干 (ViT/GeoFM)** | 高 | 跨区域泛化 +5-10% OA | ✅ Phase 3 |
| **气象数据融合** | 中 | 天气对 SAR 后向散射影响校正 | ⬜ 待定 |
| **解耦时空注意** | 中 | CropSTS 验证有效，O(T²)→O(T+S²) | ⬜ 待定 |
| **梯度检查点 + AMP** | 中 | 训练显存 -40%，速度 +50% | ✅ Phase 1 |
| **层次化场景-区域-像素** | 中 | 场景级理解 + 减少逐像素计算 | ✅ Phase 2 |
| **边界增强 (ECFEM)** | 低 | 田块边界更精确 | ⬜ 待定 |
| **OSM 先验知识** | 低 | 农田边界辅助 | ⬜ 待定 |
| **主动学习** | 低 | 智能选择标注样本 | ✅ Phase 5 |

## 七、结论

FusionCropNet V5Pro 在**多模态融合、不确定性量化、弱监督**三个维度领先所有竞品。最大短板是**无 SSL 预训练**和**无基础模型骨干**——这是 V6 的核心方向。

V6 完成后，FusionCropNet 将成为遥感作物分类领域功能最完整的框架。
